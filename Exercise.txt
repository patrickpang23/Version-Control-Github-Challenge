2. What is the major difference between Git and GitHub?
Git is a version control system that allows you to manage, keep track of, and stay on top of your work. GitHub is a place that helps you store and manage Git repositories.

3. Why do we use GitHub?
GitHub helps you store your projects into a repository for better management. It allows you to have access to it anywhere with internet connection.
